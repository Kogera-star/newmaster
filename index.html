How to use the app

  1. Enter items in the entry field using the soft keyboard or voice
  2. Tap the "ADD" button to add it to the list
  3. To cancel the entered item, tap the "Edit/Done" button
  4. Each item in the list can be moved by dragging and dropping
  5. Tap the "return" button on the soft keyboard to expand the list
  6. After inputting, tap the "SAVE" button to save
  7. Tap the "LOAD" button at the shopping destination to display the saved data
  8. After shopping, tap the "MEMO RESET" button to delete the saved data
